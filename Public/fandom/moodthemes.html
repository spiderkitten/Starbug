Rules for Use

You are free to use these anywhere, with the following caveat:

  • ★ Do not claim you made them!
  • ★ Save to your own computer, no hotlinking!

Important Info

Dreamwidth: Custom Moods are a Premium Only feature. Zip file contains all the images, and the code to use on Dreamwidth's admin console or you can create the mood theme manually on the custom mood theme page.

Livejournal: Custom Moods are not a premium feature. Unfortunately though, Livejournal no longer allows access to the admin console, you will need to add the moods manually on the custom mood theme page.
